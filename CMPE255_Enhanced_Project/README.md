# CMPE255 Final Project – Auto Data Toolkit (Enhanced: Streamlit + MLflow)

This repository contains an end-to-end ML project for CMPE255:

- Follows **CRISP-DM** (business → data understanding → prep → modeling → eval → deployment).
- Includes a **Streamlit demo app** for interactive E2E exploration.
- Includes a **reproducible MLflow training pipeline** with metrics and artifacts.
